# 디자인 가이드

> 우리집 가계부 — 디자인 시스템 명세

---

## 1. 디자인 원칙

1. **친근하고 따뜻하게** — 베이지·코랄·그린 톤. 사무적이지 않은 가정의 따스함.
2. **읽기 쉬운 큰 글씨** — 본문 17px 기본, 핵심 숫자 32~64px.
3. **명확한 위계** — 가장 중요한 숫자(잔액·예산)는 가장 크게.
4. **숨김 없는 인터랙션** — 모든 차트는 손가락으로 짚으면 정확한 숫자가 풍선으로 표시.
5. **여백을 두려워하지 않기** — 가독성 > 정보 밀도.
6. **이모지 활용** — 카테고리는 텍스트 + 이모지로 빠른 인식.

## 6. 컴포넌트 라이브러리

### 6.1 Card
- 기본: `padding: 22px 24px; border-radius: 20px; box-shadow: var(--shadow-sm);`
- `card-lg`: `padding: 28px 30px; border-radius: 28px;`

### 6.2 KPI Card
- 라벨 (eyebrow) + 아이콘 (우상단, 36×36, soft 배경)
- 큰 숫자 (clamp(22px, 3vw, 32px))
- 보조 텍스트 (전월 대비 등)

### 6.3 Button
| 변형 | 배경 | 글자 |
|---|---|---|
| Primary | `--coral` | 흰색, `box-shadow: 0 6px 14px rgba(229,118,94,.30)` |
| Default | `--surface` | `--ink` |
| Ghost   | 투명 | `--ink-2` |

크기: 기본 `12px 20px / 16px`, `btn-lg` `16px 26px / 17px`, `btn-sm` `8px 14px / 14px`.

### 6.4 Chip
- `padding: 5px 11px; border-radius: 999px; font-size: 13px;`
- 변형: `sage` `coral` `rose` `amber` `sky` (각각 soft 배경 + 진한 글자)

### 6.5 Avatar
- `sm` 28px, `md` 36px, `lg` 52px
- 원형, 흰색 한자/한글 1글자, 멤버별 컬러 배경

### 6.6 Tile (카테고리 선택)
- 4열 그리드
- 이모지 아이콘 (44×44 컬러 박스) + 라벨
- 선택 시: 코랄 테두리 + `--coral-soft` 배경

### 6.7 Bar (Progress)
- 트랙 `--bg-2`, fill `var(--coral)` 또는 카테고리 컬러
- `bar.thin` 6px, 기본 10px, `bar.thick` 14px

### 6.8 Tab Bar
- 컨테이너 `padding: 4px; background: var(--bg-2); border-radius: 14px;`
- 활성 탭: 흰 배경, `--coral-2` 글자, 작은 그림자

### 6.9 Tooltip (차트)
- 검은 배경 `var(--ink)`, 흰 글자
- `padding: 10~14px; border-radius: 10~12px;`
- 아래쪽 화살표 6px 꼭짓점
- 그림자 `0 6px 16px rgba(0,0,0,.18)`
- 호버는 `onPointerEnter` / 터치는 `onPointerDown`

### 6.10 Modal
- 오버레이 `rgba(42,37,33,0.45)` + `backdrop-filter: blur(4px)`
- 모달 최대 폭 560px, 패딩 32px
- Esc + 바깥 클릭으로 닫힘

## 7. 차트 디자인

### 7.1 막대 차트 (Monthly Bars)
- 막대 폭 18px, gap 6px
- 수입 `--sage`, 지출 `--coral`
- 비활성 막대 opacity 0.7, 활성/최근 1.0 + 살짝 그림자
- 호버 시 막대 컨테이너 배경 `--bg-2` + 위로 2px translate

### 7.2 도넛 차트
- 두께 24~28px
- 호버 segment 두께 +6, 비활성 opacity 0.45
- 가운데 표시: 비활성 시 "전체 지출 / 총액 / 조각을 눌러보세요", 활성 시 카테고리명/금액/%

### 7.3 라인 차트 (TrendArea / YearlyLine)
- 라인 2.5px, 영역 채움 opacity 0.10~0.18
- 마지막 점 강조 (큰 원 + 후광)
- 호버 시 활성 점 5~6px, 세로 가이드 점선

### 7.4 스파크라인
- 120~260px × 32~40px
- 라인만 2px, 마지막 점만 표시

### 7.5 히트맵 매트릭스
- 셀 컬러: `color-mix(in oklab, ${카테고리색} ${비율*75}%, transparent)`
- 비율 50% 이상이면 셀 글자 흰색

### 7.6 가로 스택바
- 높이 22px, 카테고리 segments
- 카드 컬러 segment 사이 2px 흰색 구분선
- 호버 시 비활성 segment opacity 0.4

## 10. 접근성 체크리스트

- [ ] 모든 텍스트 대비 비율 ≥ 4.5:1 (큰 글씨 ≥ 3:1)
- [ ] 클릭/터치 영역 ≥ 44×44px
- [ ] 키보드 Tab으로 모든 액션 도달, 포커스 링 가시화
- [ ] 색만으로 정보 전달 금지 (수입/지출은 색 + 부호 + 위치)
- [ ] 차트는 호버/터치 시 정확한 숫자 노출 (시각 의존 X)
- [ ] 글자 크기 200% 확대 시 가로 스크롤 없음
- [ ] 모달은 `role="dialog"`, `aria-modal="true"`, `aria-labelledby`

## 11. 한국어 처리 규칙

- **줄바꿈**: `word-break: keep-all` + `overflow-wrap: break-word`로 단어 중간 끊김 방지
- **금액 표시**: 기본 `1,234,567원` / 옵션 `₩1,234,567` / 한글 `123만 4,567원`
- **날짜**: `5월 12일 (월)` 또는 `2026년 5월 12일 월요일`
- **상대 시간**: `오늘`, `어제`, `n일 전` (7일 이내)
- **숫자 줄임**: 1만 이상 `n만`, 1억 이상 `n.n억`

## 12. 디자인 토큰을 코드로 (참고)

### 12.1 CSS 변수 (현재 방식)
HTML/순수 React에서는 `:root`에 정의. Tweaks 변경 시 JS로 `setProperty` 사용.

### 12.2 TypeScript 토큰 (권장 신규)
```ts
// src/design/tokens.ts
export const colors = {
  bg: '#FAF6EF',
  surface: '#FFFFFF',
  ink: '#2A2521',
  ink2: '#6F665B',
  ink3: '#9A8F82',
  coral: '#E5765E',
  coral2: '#D45F45',
  coralSoft: '#FCE4DA',
  sage: '#7FA274',
  // ...
} as const;

export const radii = { sm: 10, md: 14, lg: 20, xl: 28 } as const;
export const space = { 1: 6, 2: 10, 3: 16, 4: 24, 5: 36, 6: 56 } as const;
export const fs = {
  xs: 13, sm: 14, base: 17, lg: 19, xl: 22, '2xl': 28, '3xl': 36, '4xl': 48, hero: 64
} as const;
```

테마 전환은 `<ThemeProvider>` 또는 `data-theme="hi-contrast"` 속성으로.
