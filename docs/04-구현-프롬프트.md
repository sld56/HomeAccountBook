# 구현 프롬프트 (Claude Code 등)

> 우리집 가계부 React + TypeScript 프로젝트를 단계별로 구현하기 위한 프롬프트 모음.
> AI 코딩 어시스턴트(Claude Code, Cursor 등)에 순서대로 입력.
>
> **사용법**: 각 프롬프트 앞에 `docs/01-사양서.md`, `docs/02-디자인-가이드.md`, `docs/03-개발-가이드.md` 3개 파일을 첨부 또는 참조시키고 실행.

---

## 사전 준비

> 모든 프롬프트의 컨텍스트로 다음 자료를 먼저 제공:
> - `docs/01-사양서.md` (기능·데이터 모델)
> - `docs/02-디자인-가이드.md` (디자인 토큰·컴포넌트)
> - `docs/03-개발-가이드.md` (폴더 구조·타입·패턴)
> - 디자인 시안 HTML 파일 (`우리집 가계부.html`) — 참조용

## 프롬프트 1: 프로젝트 초기 셋업

```
첨부한 사양서·디자인 가이드·개발 가이드를 기반으로 React 18 + TypeScript + Vite 프로젝트를 셋업해줘.

요구사항:
1. Vite + React + TypeScript 템플릿으로 새 프로젝트 생성
2. 다음 의존성 설치:
   - 런타임: react-router-dom, zustand, react-hook-form, zod, date-fns
   - 개발: vitest, @testing-library/react, @testing-library/jest-dom, eslint, prettier
3. 03-개발-가이드.md의 폴더 구조 그대로 생성 (빈 폴더는 .gitkeep)
4. tsconfig.json에 path alias 설정: "@/*" → "src/*"
5. vite.config.ts에 동일한 alias 추가
6. .eslintrc.cjs + .prettierrc 기본 설정
7. Pretendard 폰트를 index.html에 CDN 링크로 추가
8. src/design/theme.css에 02-디자인-가이드.md 섹션 2~6의 모든 CSS 변수를 :root에 정의
9. src/design/global.css에 전역 reset + body 스타일 (word-break: keep-all, line-height 1.55 등)
10. src/main.tsx에서 theme.css와 global.css import

완료 후 npm run dev로 빈 페이지가 정상 뜨는지 확인.
```
